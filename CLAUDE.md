## Common Commands

To develop in this codebase, you can use the following commands:
- To build: npm run build
- To lint: npm run lint
- To run tests: npm run test
- To run a single test: npm run test -- <test_name>

## High-Level Code Architecture

The codebase is divided into the following main components:
- Frontend: Contains the React application code.
- Backend: Contains the API and server-side logic.
- Endpoints: Contains the API endpoints and their implementations.

The frontend and backend communicate through REST API calls.

## Important Files and Directories

- src/components: Contains the React components.
- src/endpoints: Contains the API endpoints.
- src/utils: Contains utility functions used throughout the codebase.
